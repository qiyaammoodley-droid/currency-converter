# Currency Converter

A simple web-based currency converter that allows users to convert between South African Rand (ZAR), United States Dollar (USD), and Bitcoin (BTC).

## Features

- Convert ZAR to USD
- Convert USD to ZAR
- Convert ZAR to BTC
- Simple and clean interface

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

This project was created to practice JavaScript logic and build a simple financial tool related to currency and trading.

The application demonstrates how user inputs can be processed and converted dynamically using JavaScript. It also shows how basic front-end technologies can be combined to build interactive web tools.

## How to Run

1. Clone or download the repository
2. Open `index.html` in a web browser
3. Enter an amount and choose currencies to convert
